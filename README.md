# Apex English Medium School 

## 🏫 About the Project
The **Apex English School Website** is a responsive and interactive web platform designed to showcase school details, academics, admissions, and contact information. It includes features for students, teachers, and administrators to access essential school resources.

## 🚀 Features
- 📌 **Home Page:** Introduction to the school with a dynamic image carousel.
- 📖 **About Us:** Information about the school's history, mission, and vision.
- 🎓 **Academics:** Details about the courses and classes taught (Kindergarten to Grade 7).
- 📩 **Contact Page:** Inquiry form with email integration.
- 📂 **Downloadable Brochure:** Users can download or view the school brochure in a new tab.
- 📸 **Image Gallery:** Display of school events and activities.

## 🛠️ Technologies Used
- **Frontend:** React.js (with Bootstrap & TailwindCSS for styling)
- **Backend:** Node.js, Express.js (if applicable)
- **Other Tools:** Git, GitHub, Vercel (for deployment)
